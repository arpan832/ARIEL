# 🚀 NASA Inspired New Tab

Transform your browser's new tab into a beautiful space-themed dashboard powered by NASA's data.

This project is a lightweight custom new tab page built with **HTML, CSS, and JavaScript**. Every time you open a new tab, you'll be greeted with stunning astronomy images fetched directly from NASA's API, creating an inspiring and distraction-free workspace.

## ✨ Features

* 🌌 Daily astronomy image from NASA
* 📸 Powered by NASA's official API
* ⚡ Fast and lightweight
* 🎨 Clean and minimal interface
* 💻 Built using only HTML, CSS, and JavaScript
* 🔧 Easy to customize

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla)
* NASA APOD API

## 🚀 Getting Started

1. Clone the repository.

```bash
git clone https://github.com/your-username/your-repository.git=
```

2. Navigate to the project folder.

```bash
cd your-repository
```

3. Open `index.html` in your browser.

If you're using your own NASA API key, replace the existing key in the JavaScript file with yours.

## 📡 API

This project uses the **NASA Astronomy Picture of the Day (APOD)** API to fetch daily space images and information.

## 💡 Inspiration

This project was inspired by NASA's breathtaking imagery and the idea that opening a new browser tab can be more than just a blank page—it can be a small reminder of how vast and beautiful the universe is.
